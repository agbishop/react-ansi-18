## Highlights

- Fast and smooth
- Foldable logs
- Automatic scrolling
- Errors detecting and fast navigator
- Regex search

## Install

```bash
npm install --save @agbishop/react-ansi-18
```

## Usage

```
import * as React from 'react'

import ReactAnsi from 'react-ansi'

function Example() {
  return (
    <ReactAnsi log={"hello world"} />
  )
}
```
